⚙️ TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════════════

PROBLEM: App won't start
SOLUTION:
  1. Ensure Python 3.7+ is installed
  2. Run with admin privileges
  3. Check PowerShell: python --version
  4. If not found, reinstall Python

PROBLEM: Read-aloud not working
SOLUTION:
  1. Check speaker/headphone connection
  2. Check Windows volume (not muted)
  3. Install pyttsx3: pip install pyttsx3 --upgrade
  4. Try different text

PROBLEM: Arduino not detected
SOLUTION:
  1. Check USB cable connection
  2. Check Arduino IDE detects it
  3. Upload firmware if not done
  4. Restart the desktop app
  5. Use Device Manager to confirm COM port

PROBLEM: Braille preview shows wrong characters
SOLUTION:
  1. Check font supports Unicode Braille (automatic)
  2. Copy preview text to check encoding
  3. Verify text input language
  4. Clear text and retype

PROBLEM: App runs very slow
SOLUTION:
  1. Close other applications
  2. Disable auto-read-aloud
  3. Update graphics drivers
  4. Restart computer
