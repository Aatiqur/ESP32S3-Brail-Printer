📊 PRINTING CAPABILITIES:
════════════════════════════════════════════════════════════════════════════

Single Line Printing:
  → Input: "আমাদের বাংলা"
  → Output: Embossed Braille on paper
  → Processing: ~5-10 seconds

Multiple Line Printing:
  → Input: "বাংলা\nব্রেইল\nপ্রিন্টার"
  → Output: 3 lines of embossed Braille
  → Processing: Sequential (5-10 sec per line)
  → Auto-detected and routed to /print-lines

Auto-Detection Logic:
  → 1 line (no \n) → /print endpoint
  → 2+ lines (with \n) → /print-lines endpoint
  → Empty lines → skipped
  → Proper spacing between lines
