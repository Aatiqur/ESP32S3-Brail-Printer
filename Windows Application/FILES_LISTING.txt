   └─ Purpose: Flask web server & Braille translation engine
   └─ Size: ~16 KB
   └─ Language: Python
   └─ Key Functions:

   └─ Lines of Code: ~400

      • Text input with visualization
      • 4 main buttons:
         - Preview Braille
         - Read Aloud
         - Print Braille (auto-detects single/multi)
         - Home Printer
      • Poetry mode checkbox
      • Status messages display
      • Beautiful animations
   └─ Framework: Tailwind CSS for styling
   └─ Icon: SVG-based icons

5. 🤖 arduin.txt (Arduino Firmware Code)
   └─ Purpose: Arduino sketch to control hardware
   └─ Size: ~3 KB
   └─ Language: C++ (Arduino)
   └─ Key Features:
      • Stepper motor control via AccelStepper library
      • Solenoid electromagnet control
      • Serial command processing

         - P (Print/Punch)

         - C (Cell shift)
         - Y (Dot row shift vertical)
         - L (Line feed)

      • Speed & acceleration settings
