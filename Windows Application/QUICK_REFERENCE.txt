🔧 SETUP (First Time Only):
════════════════════════════════════════════════════════════════════════════

Step 1: Install Python
  → Download: https://www.python.org/
  → During install: CHECK "Add Python to PATH"
  → Restart computer

Step 2: Upload Arduino Code
  → Open Arduino IDE
  → Load: arduin.txt (copy & paste into IDE)
  → Select: Board → Arduino Uno
  → Select: Port → COM port with Arduino
  → Click: Upload

Step 3: Install Packages (auto or manual)
  → Option A: Double-click INSTALL_DEPENDENCIES.bat
  → Option B: Run: pip install flask pyserial

Step 4: First Launch
  → Double-click: SINC_ROBOTICS_Braille_Embosser.bat
  → Everything works!

🆘 QUICK TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════════════

"Python not found"
  → Install Python from https://www.python.org/
  → Make sure to check "Add Python to PATH"
  → RESTART computer after installation

"Arduino offline / not detected"
  → Check USB connection
  → Make sure Arduino code is uploaded (using Arduino IDE)
  → Check Device Manager for COM port number
  → Install CH340 drivers if needed

"Nothing prints after clicking Print"
  → Check Arduino USB connection
  → Open browser console (F12) for error messages
  → Check Arduino IDE serial monitor for Arduino feedback
  → Verify stepper motor connections

"Browser won't open automatically"
  → Check port 5000 is not in use
  → Manually visit: http://127.0.0.1:5000
  → OR: http://localhost:5000

"Port 5000 in use"
  → Close other applications using port 5000
  → OR edit app.py line 329: change port=5000 to port=5001

For more help:
  → Read README.md (comprehensive)
  → Read SETUP_GUIDE.txt (detailed)

🔌 HARDWARE CONNECTIONS:
════════════════════════════════════════════════════════════════════════════

Arduino Pins (CNC Shield):
  X-axis stepper  → X pins (Pin 2, 5)
  Y-axis stepper  → Y pins (Pin 3, 6)
  Solenoid        → Pin 9
  Enable          → Pin 8
  GND             → Ground (multiple)

USB:
  Arduino USB-B   → Computer USB port

Power:
  12V-24V supply  → CNC Shield input
  GND             → Connected to Arduino GND
